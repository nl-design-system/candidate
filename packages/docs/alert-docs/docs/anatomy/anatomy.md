# Anatomie

De component bestaat uit:

1. Alert.
2. Content: container voor de inhoud van de Alert.
3. Message: container voor Header en Body.
4. Header: container voor Heading.
5. Body: container voor de inhoud van het bericht.
6. Footer: container voor eventuele acties.
7. Icon: visuele ondersteuning van doel Alert.
8. Heading: titel van het bericht.
