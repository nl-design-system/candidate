# Ordered List

Lijst waarvan de volgorde van items betekenis heeft.

## Installation

Using npm

```shell
npm install @nl-design-system-candidate/ordered-list-css
```

Using pnpm

```shell
pnpm add @nl-design-system-candidate/ordered-list-css
```

Using yarn

```shell
yarn add @nl-design-system-candidate/ordered-list-css
```
