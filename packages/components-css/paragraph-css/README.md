# Paragraph

Shows a paragraph of text.

## Installation

Using npm

```shell
npm install @nl-design-system-candidate/paragraph-css
```

Using pnpm

```shell
pnpm add @nl-design-system-candidate/paragraph-css
```

Using yarn

```shell
yarn add @nl-design-system-candidate/paragraph-css
```
