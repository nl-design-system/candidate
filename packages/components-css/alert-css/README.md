# Alert CSS

Important message informing about the user's current activity.

## Installation

Using npm

```shell
npm install @nl-design-system-candidate/alert-css
```

Using pnpm

```shell
pnpm add @nl-design-system-candidate/alert-css
```

Using yarn

```shell
yarn add @nl-design-system-candidate/alert-css
```

You can import the CSS directly from a CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nl-design-system-candidate/alert-css@1/dist/alert.css" />
```

You can import the CSS directly from the `node_modules/`

```html
<link rel="stylesheet" href="node_modules/@nl-design-system-candidate/alert-css/dist/alert.css" />
```

You can import the CSS from JavaScript

```js
import '@nl-design-system-candidate/alert-css/dist/alert.css';
```

## Design Tokens

All NL Design System components are white label. Use Alert design tokens to ensure it matches your brand styles. [https://nldesignsystem.nl/handboek/huisstijl/design-tokens/](https://nldesignsystem.nl/handboek/huisstijl/design-tokens/).

See [nldesignsystem.nl/alert/#design-tokens](https://nldesignsystem.nl/alert/#design-tokens) for a full overview. These tokens can be found in the tokens documentation [@nl-design-system-candidate/alert-tokens](https://www.npmjs.com/package/@nl-design-system-candidate/alert-tokens).
