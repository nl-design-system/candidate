# Icon React

Graphic symbol that provides visual information about an action, topic, or status.

## Installation

Using npm

```shell
npm install @nl-design-system-candidate/icon-react
```

Using pnpm

```shell
pnpm add @nl-design-system-candidate/icon-react
```

Using yarn

```shell
yarn add @nl-design-system-candidate/icon-react
```

## Design Tokens

All NL Design System components are white label. Use Icon design tokens to ensure it matches your brand styles. See [nldesignsystem.nl/icon/#design-tokens](https://nldesignsystem.nl/icon/#design-tokens) for a full overview.

These tokens are also documented in [@nl-design-system-candidate/icon-tokens](https://www.npmjs.com/package/@nl-design-system-candidate/icon-tokens).

## Other implementations

Want to use the Icon with other frameworks or vanilla JavaScript? The React component is based on
