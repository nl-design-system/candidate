# Ordered List

## Installation

Using npm

```shell
npm install @nl-design-system-candidate/ordered-list-react
```

Using pnpm

```shell
pnpm add @nl-design-system-candidate/ordered-list-react
```

Using yarn

```shell
yarn add @nl-design-system-candidate/ordered-list-react
```

## Usage

If you just want the React component

```typescript
import { OrderedList } from '@nl-design-system-candidate/ordered-list-react';
```

If you want the React component with styles

```typescript
import { OrderedList } from '@nl-design-system-candidate/ordered-list-react/css';
```
