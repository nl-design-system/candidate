# New Component CSS

{{DESCRIPTION_EN}}

## Installation

Using npm

```shell
npm install @nl-design-system-candidate/new-component-css
```

Using pnpm

```shell
pnpm add @nl-design-system-candidate/new-component-css
```

Using yarn

```shell
yarn add @nl-design-system-candidate/new-component-css
```

You can import the CSS directly from a CDN

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/@nl-design-system-candidate/new-component-css@1/dist/new-component.css"
/>
```

You can import the CSS directly from the `node_modules/`

```html
<link rel="stylesheet" href="node_modules/@nl-design-system-candidate/new-component-css/dist/new-component.css" />
```

You can import the CSS from JavaScript

```js
import '@nl-design-system-candidate/new-component-css/dist/new-component.css';
```

## Design Tokens

All NL Design System components are white label. Use New Component design tokens to ensure it matches your brand styles. [https://nldesignsystem.nl/handboek/huisstijl/design-tokens/](https://nldesignsystem.nl/handboek/huisstijl/design-tokens/).

See [nldesignsystem.nl/new-component/#design-tokens](https://nldesignsystem.nl/new-component/#design-tokens) for a full overview. These tokens can be found in the tokens documentation [@nl-design-system-candidate/new-component-tokens](https://www.npmjs.com/package/@nl-design-system-candidate/new-component-tokens).
