# New Component React

{{DESCRIPTION_EN}}

## Installation

Using npm

```shell
npm install @nl-design-system-candidate/new-component-react
```

Using pnpm

```shell
pnpm add @nl-design-system-candidate/new-component-react
```

Using yarn

```shell
yarn add @nl-design-system-candidate/new-component-react
```

## Usage

If you want the React component with styles

```jsx
import { NewComponent } from '@nl-design-system-candidate/new-component-react/css';
```

If you just want the React component and import the styles separately

```jsx
import { NewComponent } from '@nl-design-system-candidate/new-component-react';
```

## Design Tokens

All NL Design System components are white label. Use New Component design tokens to ensure it matches your brand styles. [https://nldesignsystem.nl/handboek/huisstijl/design-tokens/](https://nldesignsystem.nl/handboek/huisstijl/design-tokens/).

See [nldesignsystem.nl/new-component/#design-tokens](https://nldesignsystem.nl/new-component/#design-tokens) for a full overview. These tokens can be found in the tokens documentation [@nl-design-system-candidate/new-component-tokens](https://www.npmjs.com/package/@nl-design-system-candidate/new-component-tokens).

## Other implementations
